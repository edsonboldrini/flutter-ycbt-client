1 功能说明

Android BLE蓝牙通讯部分功能, 封装API接口供业务层调用;

1.1 主要功能

  • 蓝牙设备搜索连接;

  • 蓝牙设备设置等功能指令发送;

  • 手环健康数据解释;

  • 蓝牙设备重连管理;

1.2 主体框架

YCBTClient 类 对外API类(YCBTClient类), SDK所有接口在这里提供(连接,搜索,指令发送,数据上传等)；
BleConnectResponse 接口类 蓝牙连接状态 接口类，用于回调状态;
BleDataResponse 接口类 蓝牙指令发送，处理完成后的结果回调;
BleScanResponse 接口类 蓝牙设备搜索，结果回调;
BleRealDataResponse 接口类 蓝牙设备实时数据，结果回调(主动上报的数据,如步数,心率等);
Constants 类 常量类,数据类型,结果数据;

1.3 aar集成

  1. ycbtsdk-release.aar 放到 libs文件夹

  2. build.gradle 添加 implementation (name: ‘ycbtsdk-release’, ext: ‘aar’)

1.4 libnative_lib.so 集成

    ndk {

        moduleName "libnative_lib"

        //声明启用Android日志, 在c/c++的源文件中使用的#include <android/log.h> 日志将得到输出

        ldLibs "log"

        abiFilters "armeabi-v7a", "x86", "x86_64", "arm64-v8a"

    }

                    int optType = (int)resultMap.get("optType");  //0查询闹钟 1设备闹钟 2删除闹钟 3修改闹钟

                        int tAlarmType = (int)tAlarm.get("alarmType");

                        int tAlarmHour = (int)tAlarm.get("alarmHour");

                        int tAlarmMin = (int)tAlarm.get("alarmMin");

                        int tAlarmRepeat = (int)tAlarm.get("alarmRepeat");

                    }

                }

            }
