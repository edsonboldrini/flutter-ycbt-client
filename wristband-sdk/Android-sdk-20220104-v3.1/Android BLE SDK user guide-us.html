1 Function description

Android BLE Bluetooth communication part, encapsulating API interface for business layer to call;

1.1 Main description

  • Bluetooth device search connection

  • Bluetooth device settings and other function commands are sent

  • Interpretation of Bracelet health data

  • Bluetooth device reconnection management

1.2 Main frame

YCBTClient Class External API class (YCBTClient Class) All interface of SDK are provided here(connection, search, instruction dending, data upload, etc.);
BleConnectResponse Interface Interface Type Bluetooth connection status, used for callback status;
BleDataResponse Interface BleDataResponse interface Type Bluetooth connection status interface class, used for callback status;
BleScanResponse Interface Bluetooth device search, result callback;
BleRealDataResponse Interface Bluetooth device real-time data, result callback(actively reported data, such as steps, heart rate, etc.);
Constants Class Constant class, data type, result data;
